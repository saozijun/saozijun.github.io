Nodemon 热加载 #

开发时，修改代码后，需要频繁重启项目。nodemon配置热加载，解决了重复启动的问题。

安装 #

sh
npm install nodemon -D
1

使用 #

sh
nodemon app.js
1

或者配置在package.json #

json
"dev":"nodemon src/app",
1

这样就可以使用命令👇

sh
npm run dev
1
